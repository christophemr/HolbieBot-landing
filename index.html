About Us

What inspired me to create this project was the desire to provide immediate answers to questions that prospects or users might have. Currently, they have to send an email to a student and wait for a reply, which can be time-consuming. This project aims to streamline the process, offering instant responses and improving the overall user experience.

This project is a part of my portfolio for Holberton School. You can learn more about Holberton School here.
